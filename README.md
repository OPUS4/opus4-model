# opus4-model

Momentan dient dieses Package nur dazu, um zu testen, ob die Tests für die Implementation des Datenmodells für eine 
Datenbank ausgelagert und damit wiederverwendet werden können. Später wird vielleicht der Code für das Datenmodell 
von OPUS 4 hierher ausgelagert. Momentan befindet sich dieser Code in OPUS 4 Common (und auch noch im Framework).

Aus `Opus\Common\Document` würde dann vielleicht `Opus\Model\Document` und auch der Code in `Opus\Common\Model` würde
hierher verschoben. Noch ist aber nicht vollständig geklärt, ob das sinnvoll ist.
